# Resume Enhancer - Frontend Client

React + Vite frontend application for the Resume Enhancer SaaS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file:
   ```env
   VITE_API_URL=http://localhost:5000/api
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
